# Changelog

All notable changes to VARtificial Intelligence will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.0.0/),
and this project adheres to [Semantic Versioning](https://semver.org/spec/v2.0.0.html).

## [Unreleased]

- Academic-style README with proper ML documentation
- CONTRIBUTING.md guidelines for contributors
- MIT License file
- Python requirements.txt for ML development
- Notebooks directory for ML experimentation
- GitHub issue templates (bug reports, feature requests)
- Enhanced docstrings across all ML modules

- Updated feature engineering with proper academic references

### Added
- Initial release

- Feature engineering pipeline
- React-based web interface
- Pyodide integration for in-browser ML
- Real-time match prediction
- Interactive visualizations
