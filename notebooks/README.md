# Football Match Prediction - Exploratory Data Analysis

This directory contains Jupyter notebooks for data exploration, model experimentation, and analysis.

## Notebooks

### 1. `01_eda.ipynb` (Planned)
Exploratory Data Analysis of Premier League match data.
- Dataset overview and statistics
- Feature distributions
- Correlation analysis
- Class imbalance investigation

### 2. `02_feature_engineering.ipynb` (Planned)
Feature engineering experiments and evaluation.
- Feature importance analysis
- Polynomial feature evaluation
- Feature selection techniques

### 3. `03_model_comparison.ipynb` (Planned)
Comprehensive model comparison and hyperparameter tuning.
- Cross-validation results
- Learning curves
- Confusion matrices
- ROC-AUC analysis

### 4. `04_ensemble_methods.ipynb` (Planned)
Ensemble model development and optimization.
- Voting classifiers
- Stacking methods
- Bayesian model averaging

## Running Notebooks

```bash
# Install Jupyter
pip install jupyter

# Start Jupyter server
jupyter notebook

# Or use JupyterLab
pip install jupyterlab
jupyter lab
```

## Dependencies

```bash
pip install numpy pandas scikit-learn matplotlib seaborn
```
